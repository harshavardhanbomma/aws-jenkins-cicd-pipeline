# AWS CI/CD Pipeline using Jenkins

## Project Overview
Built an automated CI/CD pipeline using GitHub, Jenkins, and AWS EC2 to deploy a static website.

## Tools Used
- AWS EC2
- Jenkins
- Git & GitHub
- Nginx
- Linux (Ubuntu)

## Workflow
Code → GitHub → Jenkins → EC2 → Live Website

## Outcome
- Automated deployment
- Live website hosted on AWS EC2
